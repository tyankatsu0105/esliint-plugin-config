ESLint plugin for config files (ex .eslintrc.js, .stylelintrc.js)

## Usage

```bash
npm install -D eslint eslint-plugin-config-files
```

```js
module.exports = {
  plugins: ["config-files"],
  rules: {
    "config-files/xxxxx": "warn",
  },
};
```
